# Mini-DSO
This is a simple oscilloscope made with STC MCU. 

Material list:

STC8A8K64S4A12 MCU x 1 

SSD1306 OLED(SPI) x 1

Resistor: 

1W 10k x 1 

1/4W 2k x 2 

1/4W 10k x 1


Capacitor: 

47uF x 1 

0.01uF x 1


EC11 Encoder x 1 

Tact Switch x 1 

2-Pin Terminal x 1

Socket Strip: 

7-Pin x 1 

2-Pin x 1

Li-ion Battery 

5V Booster Module
